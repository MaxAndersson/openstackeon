
.. _download:

================================
Download
================================

EON is released under the `GNU General Public License version 3`_.

A read-only copy of the code can be anonymously checked out of the subversion
repository with the following command:

.. code-block:: sh

    svn co http://theory.cm.utexas.edu/svn/eon

If you are a developer you can checkout the code using the command:

.. code-block:: sh

    svn co svn+ssh://username@theory.cm.utexas.edu/svn/eon

This will fetch a copy of the latest code to a local directory named eon. 
For more information on subversion read its `online documentation`_.

The eon code can also be download directly at `tarfile`_.

.. _online documentation: http://svnbook.red-bean.com/en/1.5/index.html
.. _GNU General Public License version 3: http://www.gnu.org/copyleft/gpl.html
.. _tarfile: http://theory.cm.utexas.edu/code/eon.tgz
